# NetGuardAI

**
NetGuardAI is an IDS based on machine learning model (Random Forest).
**Attacks that NetGuardAI can detect**:
1. Port Scanning.
2. DOS Attacks.
3. DDOS Attacks.
**Libraries used**:
1. joblib
2. sklearn
3. Scapy
**Commands to run the GUI**:
python3 gui.py
On the GUI click start sniffer.
